# api

A clean Rust API project generated with [`oxgen`](https://github.com/OxgeneratorLabs/oxgenerator).

## Getting started

Run the project:

```bash
cargo run
```

Run tests:

```bash
cargo test
```

Format the code:

```bash
cargo fmt
```

Lint the code:

```bash
cargo clippy
```

Run mongodb
```bash
docker compose -f docker-compose.yml up -d --build
```

Run project
```bash
docker compose -f docker-compose.prod.yml up -d --build
```
